# Visão do Projeto

## Problema

Muitas pessoas têm dificuldade em compartilhar arquivos de maneira rápida e prática entre computadores e dispositivos. Atualmente, é comum utilizar ferramentas como Discord, WhatsApp, e-mail ou pendrives para realizar essa tarefa.

Entretanto, essas alternativas podem ser inconvenientes, pois exigem diversas etapas, podem expor informações pessoais em computadores compartilhados, dependem de dispositivos físicos que podem ser esquecidos ou perdidos e nem sempre oferecem uma experiência simples e rápida para transferências temporárias de arquivos.

## Público-alvo

Este projeto foi pensado inicialmente para estudantes universitários, grupos de trabalho e professores que necessitam compartilhar arquivos de forma rápida durante atividades acadêmicas, reuniões, trabalhos em equipe ou apresentações.

## Objetivo do Sistema

Desenvolver uma plataforma web que permita a transferência temporária de arquivos entre dispositivos de maneira simples, rápida e segura, sem a necessidade de compartilhar informações pessoais ou realizar cadastros complexos.

O sistema deverá permitir que um usuário envie um arquivo e obtenha um código ou link temporário para compartilhá-lo com outra pessoa.

## Benefícios Esperados

-Reduzir o tempo necessário para compartilhar arquivos.
-Evitar a exposição de informações pessoais.
-Eliminar a dependência de dispositivos físicos, como pendrives.
-Simplificar o processo de transferência entre dispositivos.
-Oferecer uma experiência prática e intuitiva para os usuários.
