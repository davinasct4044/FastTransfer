# Regras de negócio

## RN001 - Compartilhamento sem cadastro

O sistema deve permitir que qualquer pessoa carregue ou baixe arquivos sem a necessidade de criar uma conta ou realizar login.

## RN002 - Criação de espaço

O sistema deve permitir que qualquer pessoa crie ou entre em um espaço de compartilhamento.

## RN003 - Código único

O sistema deve criar um código único para aquele espaço que será usado para identificar e acessar o espaço.

## RN004 - Senha de acesso

O sistema deve permitir que o usuário ao criar o espaço, poderá exigir uma senha para que outros usuários acessem aquele espaço.

## RN005 - Exclusão de arquivos

O sistema deve permitir que o usuário que carregou os arquivos, possa removê-los.

## RN006 - Código inexistente

O sistema deve enviar uma mensagem de erro ao usuário caso ele tente entrar em um código inexistente.

## RN007 - Tamanho limite de um arquivo

O sistema deve limitar o usuário a colocar arquivos no limite de 100 mb, e caso o usuário exceda, proiba o carregamento do arquivo.

## RN008 - encerramento de espaço

O sistema deve permitir que o usuário que criou o espaço de compartilhamento, possa o encerrá-lo.

## RN009 - Espaço sem usuários

O sistema deve encerrar qualquer espaço que esteja sem usuários.
